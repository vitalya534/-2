# Т-Библиотека

Консольное приложение для управления личной библиотекой книг.

## Запуск

```bash
python library.py
```

## Возможности

- Добавление книг (название, автор, год, жанр, описание)
- Просмотр списка с сортировкой (по названию / автору / году) и фильтрацией (по жанру / статусу)
- Добавление/удаление книг из избранного
- Изменение статуса книги (прочитана / не прочитана)
- Просмотр избранных книг
- Удаление книг
- Поиск по названию, автору или описанию
- Рекомендации на основе любимого жанра
- Автосохранение данных в `books.json`

## Пример использования

1. Запустите программу
2. Выберите `1` — добавьте книгу "Властелин колец", автор "Дж.Р.Р. Толкин", жанр "Фантастика"
3. Выберите `2` — просмотрите список книг
4. Выберите `3` — добавьте книгу в избранное
5. Выберите `4` — отметьте книгу как прочитанную
